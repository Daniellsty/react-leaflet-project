# Train Stations Map Visualization (React + Leaflet)

## Project Overview

This is a small frontend application built with **React** that displays train stations in Germany on an interactive **Leaflet map**. Users can filter stations by city and view them on the map. The app was created as a take-home test and focuses on clean structure, React logic, UI components, and basic interaction.

---

## Features

- Interactive map powered by **React-Leaflet**
- City filter input to search stations
- Clickable markers for station locations
- (Optional) Station list with clickable items
- Test coverage using **Vitest** and **React Testing Library**

---

## Technologies Used

- **React** (frontend)
- **TypeScript**
- **React-Leaflet** for map rendering
- **Vitest** for testing
- **React Testing Library** (RTL)
- **Tailwind CSS** (optional styling)
